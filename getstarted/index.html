이 페이지에서는 공개 프레임워크와 라이브러리, 툴들의 학습을 위해서 기초 정보들을 제공합니다. 여러 웹 기술들에 대한 폭넓은 이해는 웹 프로젝트 진행시에 여러분의 선택의 폭을 넓혀주며 자신의 프로젝트 진행 방향에도 큰 영향을 줍니다. 하나의 방법만을 고집하기 보다는 다양한 학습을 해보시기 바랍니다.

웹 기술의 구분

많은 웹기술들에 대한 정보 접근을 용이하게 하기위해 본 사이트에서는 아래와 같은 구조로 웹기술들을 구분하였습니다. 현재 다루고 있는 기술들 이외에도 많은 기술들이 존재하며 차후에 계속 추가될 예정입니다.

Web Frameworks

Javascript Frameworks

웹서비스의 중추적인 역할을 하는 Business Logic들과 Back-end와의 처리를 할 수 있도록 제공되는 Framework들입니다. 일부 프레임워크들은 Back-end logic까지 포함하여 Full-stack JS Framework를 구성하는 경우도 있습니다.

  • laravel 2018.12.02

    laravel은 php 기반의 프레임워크입니다.

  • vue.js 2018.12.01

    vue는 효과적인 프론트엔드 개발을 지원하는 프레임워크입니다.

  • React 2018.11.25

    페이스북이 만든 자바스크립트 프레임워크(라이브러리)인 React를 Comment를 사용한 예제와 함께 시작해본다.

  • react native 2018.11.20

    react native는 효과적인 모바일 앱 개발을 지원하는 프레임워크입니다.

  • Express.js 2018.08.04

    Express.js는 노드로 구현한 웹서버 라이브러리 입니다. Express.js의 구조와 사용법에 대해 알아봅니다.

  • electron 2017.12.29

    electron은 JS 기반의 데스크탑 어플리케이션 개발 프레임워크입니다.

  • typescript 2017.12.28

    typescript

  • Hapi.js 2016.07.14

    Hapi.js는 노드로 구현한 웹프레임웍 입니다. Hapi.js의 구조와 사용법에 대해 알아봅니다.

  • Meteor.js 2015.09.30

    Meteor.js는 Node기반의 풀스택 프레임워크입니다.

  • ExtJS5 Getting Start 2014.09.14

    ExtJS는 기업환경 적합한 최적의 성능과 수많은 컴포넌트를 내장하고 있는 JS프레임웍이다. CS애플리케이션의 모든 컴포넌트를 웹브라우저로 옮겨왔고 자체 클래스 시스템과 개발, 빌드, 실행 등을 도와주는 도구들을 모두 갖춘 토탈프레임웍으로 각광 받고 있다

  • AngularJS Getting Start 2014.09.06

    구글이 만든 자바스크립트 프레임워크인 AngularJS를 시작해본다. 간단한 사용자 목록을 확인하는 프로그램을 통하여 AngularJS의 기본 기능을 이용해 보면서 AngularJS에 대하여 간단히 살펴보겠다.

  • Backbone.js 시작하기 2014.09.06

    Model-Collection-View 기반으로 구현하는 Javascript 웹 애플리케이션 프레임웍

Tools

웹 프로젝트들을 보다 쉽고 빠르게 설치/관리할 수 있게 도와주는 도구들입니다.

  • docker 2018.11.30

    docker는 컨테이너 기반의 가상화 서버입니다.

  • Webpack 2016.11.04

    자바스크립트 모듈 번들러인 웹팩(Webpack)에 알아봅니다.

  • Jasmine Javascript Testing 2015.09.30

    웹프론트엔드를 개발하다보면 특히나 많은 테스트를 진행해야하는데, 진행이 쉽지가 않습니다. Jasmine은 자바스크립트 코드 테스트를 위한 행위주도개발프레임워크입니다.

  • Yeoman, Kickstart your project 2014.09.04

    웹프론트엔드는 각각의 프로젝트별로 다양한 기술을 사용하는 환경을 가지고 있습니다. 그중에서 대표적인 각 기술별로 베스트 프렉틱스(Best Practices) 와 툴이 포함된 프로젝트를 빠르게 생성해주는 도구입니다.

  • Bower 2014.09.04

    웹 프론트엔드 패키지 매니저인 Bower 에 대해서 알아봅시다. Bower 는 간단한 명령을 통해서 라이브러리/프레임워크들의 의존성을 관리할 수 있습니다.

  • Grunt and Gulp 2014.08.20

    빌드시스템인 Grunt 와 Gulp 에 대한 소개와 사용법에 대해서 알아봅니다. 현재 개발중인 프로젝트에 빌드 시스템을 적용하여 여러가지 빌드에 관련된 작업을 간단하게 처리 할 수 있습니다.
